## Применяемые технологии

- Node.js
- Express
- Postman
- MongoDB
- Mongoose

## Запуск:

- `npm i` - устанавливаем зависимости
- `npm run dev` - для запуска сервера с hot-reload
- `npm run start` - для запуска сервера в продакшн-режиме
